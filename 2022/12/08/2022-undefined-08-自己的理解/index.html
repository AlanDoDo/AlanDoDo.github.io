Free Programming Books

发表于2022-12-08|更新于2022-12-23|技术分享
|阅读量:

Free Programming Books 是一个提供免费编程书籍的网站。网站无需注册登录，所有数据文档均可以直接下载。
网站内有各类编程语言的教程，包括但不限于 Java、C、C++、C#、CSS、HTML5、JavaScript，还有数据结构和数据库等。

Replit
Replit 一款免费的、在线协作式、基于浏览器的 IDE，可使用50多种编程语言进行编码，编译，运行和托管。从 Python 到 C++，再到 HTML 和 CSS，我们可以在一个平台上学习并使用任何语言进行编码。
Replit 支持邮箱注册、Github 登录和 Google 登录，注册之后即可创建自己的在线项目，你可以在任何有浏览器的地方编写程序。通过 0 手动设置就可以导入、运行和协作处理数百万个 GitHub 存储库。实时协作，可以直接邀请朋友、队友和同事加入代码编辑，在其中会看到协作者的光标。

ScrapingBee
ScrapingBee 是一个网络爬虫工具，可以解决代理、验证码等各种爬虫问题。
Scrapingbee 可以通过以下三种方式来抓取数据：
1.常规 Web 抓取，例如房地产抓取、价格监控、提取评论；
2.抓取搜索引擎结果页信息，用于 SEO、关键字监控或反向链接检查；
3.增长黑客，包括潜在客户生成、提取联系信息或社交网络信息。

大屏数据展示模板
包含各类大屏数据可视化模板分享，在 gitee 上已有 16.8k star。涉及智慧交通、医疗、城市、政务、物流、气象、电商、运维等等领域。页面美观，有各式各样的统计图，配色也十分的出彩。

HanLP
HanLP 是一个面向生产环境的多语种自然语言处理工具包，其基于 PyTorch 和 TensorFlow 2.x 双引擎，目标是普及落地最前沿的NLP技术。
HanLP 具备功能完善、精度准确、性能高效、语料时新、架构清晰、可自定义的特点。借助世界上最大的多语种语料库，HanLP支持包括简繁中英日俄法德在内的104种语言上的10种联合任务：分词、词性标注、命名实体识别、依存句法分析、成分句法分析、语义依存分析、语义角色标注、词干提取、词法语法特征提取、抽象意义表示。
